Planned Features

Core Language Features

  • Arrays and basic data structures
  • String manipulation functions
  • Loop constructs (for, while)
  • Function parameters and return values
  • Error handling with 'perkele' as try-catch mechanism

Standard Library

  • File I/O operations
  • Basic math functions
  • Random number generation
  • Date and time handling

Developer Experience

  • REPL (Interactive shell)
  • Package manager named "sauna"
  • VSCode syntax highlighting extension
  • Better error messages with Finnish translations

Advanced Features

  • Multi-threading support with "kahvi" keyword
  • Network programming primitives
  • FFI support for more languages beyond C
  • Module system for code organization
