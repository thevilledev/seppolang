What is SeppoLang?

SeppoLang, or Seppo for short, is a minimalist programming language inspired by Seppo's around the world. It uses the keyword 'seppo' for various operations, making it uniquely Finnish and deliberately confusing.

Basic Syntax

The keyword 'seppo' can be used to print values when used with an expression: 

fn seppo() {
    x = 42
    seppo x  // prints 42

    // seppo as a conditional (if-else)
    seppo x > 40 {
        y = 10
        seppo y  // prints 10
    }
    perkele {
        y = 5
        seppo y  // prints 5
    }
}

Advanced Features

Inline C

You can write inline C code using the special ceppo block. You can define C functions outside main and call them from SeppoLang: 

// Define C functions outside seppo()
ceppo {
    int add_numbers(int a, int b) {
        return a + b;
    }
}

fn seppo() {
    // Call C function from SeppoLang
    x = 10
    y = 20
    result = add_numbers(x, y)
    seppo result // prints 30
}

Key Features

  • 'seppo' works as a print statement
  • 'seppo' can be used for conditional statements
  • 'perkele' serves as the 'else' keyword
  • Seamless C integration with 'ceppo' blocks
